# MapleSky

A MapleStory v83 private server — pre-Big Bang nostalgia.

## How to Play

1. **Download** the latest client from [Releases](https://github.com/octobrium/MapleSky/releases)
2. **Extract** to `C:\Nexon\MapleStory`
3. **Run** `MapleSky-WINDOW.exe`
4. **Register** at the login screen — pick any username and password, your account is created automatically

## Server Info

| | |
|---|---|
| **Version** | v83 (Pre-Big Bang) |
| **Rates** | 3x EXP / 2x Meso / 3x Drop / 5x Boss |
| **Quests** | 1x (Vanilla) — +1 fame per quest completed |
| **Multi-client** | Unlimited |
| **Classes** | All v83 explorers |

## Troubleshooting

- **Game crashes on launch:** Right-click `MapleSky-WINDOW.exe` → Properties → Compatibility → check "Run in compatibility mode" → select Windows XP SP3
- **Can't connect:** Make sure you extracted to `C:\Nexon\MapleStory` (exact path required)
- **Black screen:** Try running as Administrator

## Requirements

- Windows 10 or 11
- No other MapleStory installation needed — the download includes everything
